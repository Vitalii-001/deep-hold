# Deep Hold — дизайн-документ

**Дата:** 2026-07-10
**Статус:** затверджено користувачем
**Тип продукту:** HTML5 idle/incremental гра для веб-порталів (CrazyGames, Poki)

## Концепція

*Ти — король гномів. Копай углиб, будуй імперію, вари ель і не розбуди те, що спить на дні.*

Idle-гра з темою підземної імперії гномів. Вісь прогресу — глибина копання. Унікальна механіка — цикл еля: гноми потребують еля для ефективної роботи. Мова гри — англійська (глобальний ринок порталів).

**Робоча назва:** Deep Hold (може змінитись перед релізом).

## Цільова платформа і бізнес-модель

- **Формат:** HTML5, білд < 5 МБ, сесія 3–10 хв, керування мишкою/тачем.
- **Шлях релізу:** itch.io (фідбек) → правки балансу → CrazyGames (revenue share ~50% з реклами).
- **Монетизація:** rewarded ads через CrazyGames SDK («Благословення предків ×2 на 4 год», «безкоштовна бочка еля»). IAP — після перших метрик, не в v1.
- **Ключова метрика:** retention (повернення гравця наступного дня). Гачок повернення — офлайн-прогрес.
- **Апгрейд-шляхи при успіху:** Poki → Steam-версія → ексклюзивна ліцензія.

## Core loop

1. Клік кайлом → видобуток каменю (активна фаза перших хвилин).
2. Наймання гномів-працівників → пасивний видобуток.
3. Будівництво споруд → переробка ресурсів (кузня плавить руду, броварня варить ель).
4. **Цикл еля:** гноми п'ють ель і працюють швидше; без еля — страйк, видобуток падає. Другий ресурсний цикл + гумор.
5. Копання вглиб → нові шари з новими ресурсами, апгрейдами, палітрою і ризиками.

## Прогрес

- **Шари глибини (v1 — 6):** ґрунт → камінь → залізо → золото → самоцвіти → руїни древніх. Натяк на «древнє зло» внизу (ризик/винагорода: швидке копання = шанс обвалу).
- **Милстоуни глибини:** цілі на сесію («Глибина 500 м: знайдено покинуту залу предків»).
- **Ресурси:** камінь, руда, злитки, золото, самоцвіти, ель.
- **Працівники (v1 — 4 типи):** шахтар (видобуток), коваль (переробка руди у злитки), бровар (виробництво еля), розвідник (прискорює просування вглиб / відкриття нових шарів).
- **Споруди (v1 — 6):** шахтний ствол, плавильня, кузня, броварня, велика зала, храм предків.
- **Апгрейди:** 20–25 у v1.
- **Престиж («Заснувати нове королівство»):** скидання прогресу за руни предків з перманентними бонусами — **v1.5, не v1**.

## Технічний стек

- **Vite + React + TypeScript + Zustand** — UI на React-компонентах, ігровий стан у Zustand, тік-система на requestAnimationFrame.
- **Збереження:** localStorage; офлайн-прогрес — розрахунок по timestamp («Поки тебе не було, гноми накопали 12,4K каменю»).
- **Графіка:** SVG-іконки з game-icons.net (ліцензія CC BY — вказати авторів у credits), CSS-анімації, канвас-шар для візуалізації шахти в розрізі (гноми-пікселі по рівнях). AI-генерація — для 1–2 великих артів (обкладинка, фон).
- **CrazyGames SDK:** реклама + аналітика.
- **Звук:** мінімум — кирка, монети, звук гнома після еля.

## Скоуп v1

**Входить:** клік-видобуток, 4 типи працівників, 6 споруд, цикл еля, 6 шарів глибини, 20–25 апгрейдів, офлайн-прогрес, збереження/завантаження, заглушки rewarded ads, балансна таблиця (окремий конфіг для тюнінгу).

**Не входить (v1.5+):** престиж, події/квести, розширений звук, локалізації, IAP.

## Таймлайн (10–20 год/тиждень)

| Тижні | Етап |
|---|---|
| 1–4/5 | Розробка v1 |
| 5 | Реліз на itch.io, збір фідбеку |
| 6–7 | Правки балансу (баланс = 80% успіху idle-гри) |
| 7+ | Подача на CrazyGames |

## Ризики

- **Баланс:** головний ризик жанру; мітигація — балансна таблиця в конфігу + ранній фідбек з itch.io.
- **Модерація CrazyGames:** вимоги до якості/розміру; мітигація — тримати білд < 5 МБ, дотримуватись їхніх гайдлайнів SDK з першого дня.
- **Дисперсія доходу:** перша гра — це десятки-сотні $/міс; проєкт також слугує розвідкою пайплайну порталів для наступних ігор.
