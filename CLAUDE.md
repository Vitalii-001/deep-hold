# Deep Hold

HTML5 idle/incremental гра «підземна імперія гномів» для веб-порталів (CrazyGames, Poki). Комерційна мета — revenue share з реклами на порталах.

## Статус

- **Дизайн затверджено:** `docs/superpowers/specs/2026-07-10-deep-hold-design.md` — єдине джерело правди по скоупу, механіках і стеку. Читати перед будь-якою роботою.
- **Наступний крок:** створити покроковий план імплементації (скіл superpowers:writing-plans на основі спеки), далі — виконання плану.
- Коду ще немає — проєкт не заскафолджений.

## Ключові рішення (деталі в спеці)

- Стек: Vite + React + TypeScript + Zustand; тік-система на requestAnimationFrame; збереження в localStorage з офлайн-прогресом.
- Мова гри — англійська. Спілкування з користувачем — українською.
- Білд < 5 МБ (вимога порталів).
- Баланс — в окремому конфігу, не хардкодити в компонентах (тюнінг балансу — головна ітераційна робота жанру).
- Іконки: game-icons.net (CC BY — вести список авторів для credits).
- Престиж, IAP, локалізації — свідомо поза скоупом v1.

## Контекст користувача

- Фронтенд-розробник, впевнений у React/Next.js і Stripe; темп — 10–20 год/тиждень.
- Роль користувача в розробці: ревью, плейтест і рішення по балансу/відчуттях геймплею.
